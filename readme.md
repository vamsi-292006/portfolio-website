# Vamsi Krishna Pedda | Premium Developer Portfolio

A highly animated, modern, and fully responsive personal portfolio website designed to showcase my journey as a Competitive Programmer, Machine Learning enthusiast, and B.Tech ICE student at NIT Jalandhar.

## ✨ Features

* **Premium UI/UX:** Built with a modern glassmorphism aesthetic and a sleek dark-mode-by-default theme.
* **Highly Animated:** Buttery-smooth, scroll-triggered reveal animations powered by GSAP.
* **Interactive Elements:** * Custom magnetic cursor that tracks mouse movement.
  * 3D tilt effects on project cards on hover.
  * Interactive particle background in the Hero section.
  * Dynamic typing text effect.
* **Light/Dark Mode:** Seamless theme toggling that updates the entire UI and particle colors.
* **Working Contact Form:** Fully functional contact form integrated with Formspree (no backend required) with loading states and success/error handling.
* **Fully Responsive:** Mobile-first design that looks perfect on desktops, tablets, and smartphones, including a custom mobile slide-in menu.

## 🛠️ Tech Stack

This project was built intentionally without heavy frameworks to ensure maximum performance, relying on raw JavaScript and powerful libraries.

* **Markup & Styling:** HTML5, CSS3, [Tailwind CSS](https://tailwindcss.com/) (via CDN)
* **Logic:** Vanilla JavaScript (ES6+)
* **Animations:** [GSAP](https://greensock.com/gsap/) (ScrollTrigger)
* **Visual Effects:** [Particles.js](https://vincentgarreau.com/particles.js/), [Vanilla-tilt.js](https://micku7zu.github.io/vanilla-tilt.js/)
* **Icons:** FontAwesome
* **Backend Integration:** [Formspree](https://formspree.io/) (for email handling)
